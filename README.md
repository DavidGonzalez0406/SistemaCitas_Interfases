# SistemaCitas_Interfases
Tarea 3: Maquetado de sistema de citas

Sistema de Citas (Interfaces)
Este es un proyecto de escritorio desarrollado en Java utilizando JavaFX y gestionado con Maven. Está diseñado para el manejo de interfaces de un sistema de citas.

🛠️ Tecnologías Utilizadas
Lenguaje: Java 24.
Framework de UI: JavaFX 21.
Gestor de Dependencias: Maven.
Componentes: JavaFX Controls y FXML para el diseño de interfaces.

🚀 Requisitos Previos
Antes de ejecutar el proyecto, asegúrate de tener instalado:
JDK 24 (o superior).
Maven instalado en tu sistema.
